# 부루마불 (Blue Marble) 웹 보드게임

부루마불과 유사한 웹 기반 보드게임입니다.

## 게임 방법
- 2~4명이 주사위를 굴려 보드를 돌며 부동산을 매매하는 게임
- 봇 플레이 지원 (1~3명의 AI 봇과 대전)
- 더블 주사위 시 추가 턴

## 보드 구성 (40칸)
출발 → 타이베이 → 황금열쇠 → 베이징 → 마닐라 → 제주도 → 싱가포르 → 황금열쇠 → 카이로 → 이스탄불 → 무인도 → 아테네 → 코펜하겐 → 황금열쇠 → 스톡홀름 → 베른 → 베를린 → 황금열쇠 → 오타와 → 부에노스아이레스 → 사회복지기금 → 상파울루 → 시드니 → 황금열쇠 → 부산 → 하와이 → 로마 → 파리 → 황금열쇠 → 런던 → 세계여행 → 뉴욕 → 서울 → 황금열쇠 → 도쿄 → 콜롬비아호 → 황금열쇠 → 베니스 → 황금열쇠 → 출발
